# Notation3 (N3) Specification

## Notation3 Dialect

Notation3 (N3) is an assertion language and logic dialect that extends standard RDF. While RDF is restricted to simple triple assertions, N3 introduces:
* **Variables**: Prefixed with `?` (e.g., `?x`), representing logical variables.
* **Logical Implication**: Declared with `=>` (forward implication) and `<=` (backward implication).
* **Quoted Graphs**: Wrapped in `{}` braces, representing statements as first-class terms.
* **Lists**: Wrapped in `()` parenthesis, representing ordered sequences.
* **Built-ins**: Procedural functions for arithmetic, string formatting, and list processing.

---

## Implementation in Roxi

Roxi implements a complete N3 parser and compiler. Implication rules are evaluated inside the forward chainer, enabling expressive rule-based reasoning over RDF datasets.
