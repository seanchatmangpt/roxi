# Dialect Conformance Metrics

## Conformance Goals

Roxi tracks conformance status across all supported dialects:

* **SPARQL 1.1**: W3C query evaluation and update tests.
* **Datalog**: Custom Datalog core test suite.
* **N3**: EYE Reasoner N3 community test corpus.
* **SHACL**: W3C data-shapes test suite.
* **ShEx**: shexTest conformance suite.
